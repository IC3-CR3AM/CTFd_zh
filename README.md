>注意：该版本只支持python3，建议使用python3.6左右，更高版本可能会出现问题



## CTFd_zh
 * 基于CTFd原有平台模板进行优化
 * 加入选择题选择题平台
 * 整合和论坛与选择平台

## Ubuntu 安装方法
 * 安装git
   * sudo apt-get install git
 * 安装pip
   * sudo apt-get install python-pip
 * 安装Flask
   * sudo pip install Flask
 * 从github上获取CTFd_zh
   * git clone https://github.com/totalo/CTFd_zh.git
   * cd CTFd_zh
   * sudo ./prepare.sh
   * sudo python serve.py
